# HOLA CLAUDIA

QUÉ GUAPA ERES

Esta web ya no vive en WordPress. Después de años de hosting de pago, mantenimiento y parches de seguridad, jmbravo.com ha pasado a ser un sitio estático: texto plano en Markdown, generado con Hugo y publicado en GitHub Pages.

¿Qué cambia para quien lee? En principio, nada que se note —el contenido es el mismo—, salvo que la web debería cargar más rápido. Lo que cambia de verdad es todo lo que no se ve: sin base de datos, sin plugins, sin facturas anuales.

Esta misma nota es la primera entrada de una sección nueva, «Novedades», pensada para apuntes breves como este, separados de los artículos más extensos sobre oficio de escritura.
